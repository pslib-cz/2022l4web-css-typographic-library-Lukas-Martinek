Implementation

portfolio.css is typhographic CSS library which you can download for free. It was created for school project. If you have finished code and dont want to waste time, you can use my project. portfolio.css is styling headings, text markups, lists, tables, buttons and images.

All you need to do is download CSS file and then connect it to your already written code.
